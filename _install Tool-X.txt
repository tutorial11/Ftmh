-bagi agan yang belum punya aplikasinya bisa di download dan install di "Play Store"


























- setelah mendonwnload dan selesai meng install langsung buka aja termux nya.























- sekarang ke tahap penginstallan
   Oke langsung aja cekidot...)
 






























                  Buka termux dan  Ketikan

$ pkg install root-repo                 (ENTER)
$ pkg install unstable-repo         (ENTER)
$ pkg install x11-repo                   (ENTER)
$ apt update && apt upgrade    (ENTER)
$ apt install git                               (ENTER)

Tanda "$" gak usah di ketik gan )
 Jika ada pilihan (Y/T) ketik aja Y lalu enter






















-Kita download dulu file nya dengan mengetik

$ git clone https://github.com/Rajkumrdusad/Tool-X    (ENTER)


















-Sekarang kita buka file yang tadi udah di    download dengan mengetikan

$ ls                         ( ENTER )
$ cd Tool-X           ( ENTER )

 tanda "ls" menggunakan ( L kecil bukan i besar )





































- lanjut ke perintah untuk merubah izin akses file atau direktori, dengan mengetik 
$ ls                                      ( ENTER )
$ chmod 777 install.aex  ( ENTER )
$ ls                                      ( ENTER )

Jika berhasil maka install.aex akan berubah menjadi hijau.


































- install.aex dengan mengetik

$ ./install.aex       (ENTER)
 
  Jika ada pilihan (Y/T) ketik saja Y  (ENTER)































- jika sudah, tutup aplikasi termux dan untuk menjalankan nya lagi ketikan : 

$ Tool-X

  Maka akan muncul beberapa pilihan

[0] Install all tools. (total 264 tools) 
[1] Show all tools. (almost 264 tools) 
[2] Tools category.
[3] Termux OS.
[4] Update Tool-X.
[5] About us.
[x] For Exit.

- Ketik 0 untuk install semua tools, total 264    tools

- ketik 1 untuk melihat semua tools. Dan memilih tools mana saja yang ingin agan install 

       Dan seterusnya :)




























- karna saya tidak akan menginstall semuanya, jadi saya ketikan 1 lalu enter untuk memilih tools yang ingin saya install.




























- saya akan menginstall tools LITESPAM dengan memilih nomor 209 
- Ketikan 209 lalu (ENTER)




































- setelah selesai maka akan muncul cara menjalankan tools yang tadi agan pilih









































- dan untuk menjalankan tools yang tadi agan pilih dan install, agan harus keluar dari tool-X dengan mengetik
- 00 (ENTER) untuk kembali dan ketik
-  × (ENTER) lalu ketik Y (ENTER).


































- dan sesudah keluar, agan tinggal jalankan tools yang tadi sudah di pilih dan di install.. 






























- setiap tool yang agan install cara menjalankan tool nya mungkin beda beda, jadi setelah install di Tool-X baca dulu cara menjalankan.nya :)
































-karna saya memilih LITESPAM jadi cara menjalankannya tinggal mengetik

$ cd (ENTER)
$ ls (ENTER) untuk melihat file yang sudah di download tadi.
$ cd LITESPAM         (ENTER)
$ ls                               (ENTER)
$ sh LITESPAM.sh    (ENTER)



































- Dan tadaaaa tool LITESPAM yang tadi saya install sudah bisa di gunakan.
